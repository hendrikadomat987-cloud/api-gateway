Build a Node.js Test Engine for API Gateway + n8n workflows.

Requirements:
- Use axios
- Implement reusable CRUD test runner
- Follow pattern: Arrange → Act → Assert → Cleanup
- Support multi-tenant JWT auth
- Structure:
  tests/core (apiClient, testRunner)
  tests/<entity>/crud.test.js
- Tests must:
  - create entity
  - update entity
  - validate response
  - delete entity
- Must be reusable for all entities (customer, request, etc.)
- Output clean logs and throw errors on failure
