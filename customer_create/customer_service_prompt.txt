You are an expert backend engineer specializing in Node.js, n8n
workflows, and SaaS architectures.

We are building a multi-tenant SaaS platform for AI voice agents. The
infrastructure is already set up (Docker, n8n, Redis, Supabase
PostgreSQL, API Gateway with JWT).

Your task is to generate a clean, production-ready n8n workflow for a
“Customer Service”.

Context

-   Requests come through an API Gateway (Node.js)
-   JWT authentication is already handled upstream
-   The request is forwarded to n8n via webhook
-   Multi-tenant architecture (tenant_id required)
-   Database: Supabase PostgreSQL with Row Level Security (RLS)

Input (Webhook)

POST /webhook/customer

Example payload: { “name”: “Max Mustermann”, “phone”: “123456789”,
“email”: “max@test.de” }

Headers: - x-user-id - x-original-url - authorization (already validated
upstream)

Requirements

Build a complete n8n workflow with the following structure:

1.  Webhook Node
2.  Set Node
3.  Validation Node
4.  Database Insert
5.  Respond Node

Constraints

-   Keep data flat in $json
-   Avoid undefined/null mapping errors

Output

Provide: 1. Full n8n workflow JSON export 2. Explanation of each node
